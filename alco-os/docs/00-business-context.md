# 00. 事業背景

## 合同会社アルコ

千葉県館山市を拠点に、以下の事業を行う。

| 事業 | 内容 |
|---|---|
| 館山ジビエセンター | ジビエ食肉処理・加工・流通（食品事業としてのトレーサビリティが生命線） |
| R.O.K.A. | 館山ジビエラボ。研究・飲食・地域拠点。現在リノベーション中 |
| 南房総里山未来拠点 | 里山保全・自然資本・地域活性 |
| 自然資本コンサルティング | 自然共生サイト認証支援、TNFD、ネイチャーポジティブ、30by30、生物多様性クレジット |
| 補助金・連携 | 補助金申請、行政連携、企業連携、地域団体連携 |

## 代表（沖浩志）の特性と課題

- **強み**: 現場、観察、関係構築（BNI・同友会・YEG・行政・企業）
- **補完したい領域**: デスクワーク、書類仕事、事務処理
- 移動中・現場後・会議後の音声メモに大量の業務情報が眠っている

## ALCO OS の存在理由

単発アプリを乱立させると、データが散り、後で全部作り直しになる。
ALCO OS は「同じデータベース・同じ権限・同じAI処理基盤」の上に
事業モジュールを段階的に足していく **業務オペレーティングシステム** である。

> ALCO OS の本丸は「代表の思考・現場・事業・自然資本を、
> 事務処理と経営判断に変換する基盤」。

## 既存資産（重要）

リポジトリルートには **ジビエ基幹システム（静的HTML + Supabase）が本番稼働中**。
individuals（個体）、hunters（捕獲者）、staff / attendance（勤怠）、
products / product_movements（完成品在庫）などの実データを持つ。
ALCO OS はこれを壊さず共存し、段階的に統合する（docs/09-gibier-integration.md）。
